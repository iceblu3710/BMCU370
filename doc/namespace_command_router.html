void CommandRouter::Run ( )

Main execution loop for the Router.

Checks for new packets from the underlying protocol (SerialCLI, KlipperCLI, or BambuBus). If a packet is ready, it identifies the type (if Bus mode) and calls Route. Also manages the "Offline" timeout logic (3 seconds).
