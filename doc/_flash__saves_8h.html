bool Flash_saves ( void * buf,
uint32_t length,
uint32_t address )
extern

Save a buffer to Flash Memory.

Erases the required number of pages starting at address and programs the data. Reloads IWDG during the process to prevent watchdog reset.

Parameters
bufPointer to the source data.
lengthLength of data in bytes.
addressDestination Flash address (Page Aligned recommended).
Returns
true Success.
false Fail.
